# Task Manager API (Laravel)

### Project Overview
This is a RESTful API Task Manager built with Laravel, featuring:
* **Authentication:** Secure Login/Register using Laravel Sanctum.
* **Profiles:** User profile management.
* **Roles & Permissions:** Access control and user permissions.
* **Task CRUD:** Create, Read, Update, and Delete tasks.
* **Favorites:** Ability to add tasks to a favorites list.

### Tools Used
* Framework: Laravel
* Testing: Postman
* Database: MySQL
